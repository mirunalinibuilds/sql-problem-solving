# 1% Better SQL

A collection of SQL solutions that I solve while preparing for coding interviews and improving my database skills.

## Why This Repository?

I believe in becoming 1% better every day.

This repository documents my SQL problem-solving journey through LeetCode and other interview-style questions. The goal is not just to solve problems, but to understand the underlying concepts and write clean, efficient SQL queries.

- WHERE & Filtering
- ORDER BY
- GROUP BY
- HAVING
- JOINS
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- Case Statements
